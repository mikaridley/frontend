# ‚ö™ Marshmello

**Marshmello** is a lightweight, intuitive, and "soft" project management tool designed for teams who want to keep their workflow organized and stress-free. Named for its smooth UI and flexible structure, Marshmello turns complex project management into a simple, sweet experience.

---

## ‚ú® Features

- **Dynamic Kanban Boards:** Drag-and-drop tasks from "To Do" to "Done."
- **RTL Support:** Full Right-to-Left layout compatibility, optimized for Hebrew and Arabic.
- **Task Checklists:** Built-in sub-tasks to ensure no detail is missed.
- **Smart Columns:** Pre-configured columns for **Icebox**, **Blocked**, and **QA**.
- **Real-time Sync:** Collaborate with your team instantly.
- **Soft UI Design:** A clean, minimal aesthetic that reduces work-related eye strain.

## üìã Recommended Board Layout

Marshmello works best when you follow our "Smooth Flow" organization:

| Column          | Purpose                                               |
| :-------------- | :---------------------------------------------------- |
| **Resources**   | Links, brand assets, and API documentation.           |
| **Icebox**      | Ideas and "later" features not yet in the sprint.     |
| **To Do**       | Tasks ready for immediate development.                |
| **In Progress** | Tasks currently being coded or designed.              |
| **Blocked**     | Tasks stuck due to dependencies or external feedback. |
| **QA / Review** | Tasks requiring testing or code review.               |
| **Done**        | Tasks that are 100% completed and toasted.            |
